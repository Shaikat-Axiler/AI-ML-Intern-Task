# AI/ML Internship Task Repository

## Overview

This repository contains all projects, assignments, and deliverables completed as part of a structured 4-week AI/ML Internship Program. The program focuses on practical implementation of machine learning systems, progressing from Python fundamentals to production-ready AI applications.

---

## Internship Structure

### Week 1: Python Foundations & LLM Basics

Key topics:
- Python fundamentals (data structures, control flow, OOP)
- File handling (JSON, CSV)
- Git workflow and version control
- Transformer basics and tokenization
- Hugging Face ecosystem and inference

Project:
LLM Explorer Toolkit
- Compare outputs from multiple language models
- Experiment with prompt engineering techniques
- Save and evaluate generated responses

---

### Week 2: Data Pipelines & Model Understanding

Key topics:
- Data collection and preprocessing
- Dataset creation for language models
- Tokenization techniques (BPE, WordPiece, SentencePiece)
- Evaluation metrics (Perplexity, BLEU, ROUGE)

Project:
Dataset Builder & Model Profiler
- Multi-source data ingestion (CSV, JSON, APIs)
- Data cleaning and validation pipeline
- Model profiling (memory usage, inference speed)

---

### Week 3: Fine-Tuning & Alignment

Key topics:
- Supervised Fine-Tuning (SFT)
- Parameter-efficient tuning (LoRA, QLoRA)
- Direct Preference Optimization (DPO)
- Training configuration and optimization

Project:
Domain-Specific Model Tuner
- Fine-tune models for a specific domain
- Evaluate performance improvements
- Export and deploy locally

---

### Week 4: Agentic AI & Production Deployment

Key topics:
- Retrieval-Augmented Generation (RAG)
- Vector databases (FAISS, Chroma)
- Agent frameworks and tool usage
- API development and Docker deployment

Project:
Intelligent Document Assistant
- Document-based question answering system
- Conversational interface with memory
- REST API with deployment support
